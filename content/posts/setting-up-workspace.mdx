Welcome to my new personal blog. I wanted a space to document my learnings, but I also wanted it to feel like *home* — an homage to the terminal interfaces and retro systems I love.

### The Stack

I built this using Next.js with the App Router. It gives me the flexibility of React with incredible performance.

<GitHubPreview repo="vercel/next.js" />

For styling, I'm heavily relying on Tailwind CSS and CSS variables to handle the complex theme switching between Terminal, Cyberpunk, and Retro modes.

It's been a fun experiment in design constraints. Expect more posts soon on memory optimization, scripting, and interesting tools I find in the wild.
